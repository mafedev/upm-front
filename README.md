# CTB-UPM - Sistema de Gestión de Dispositivos

Este repositorio contiene dos proyectos principales que forman parte del sistema:


## 1. Android Front (android_front_upm)

Aplicación móvil desarrollada en Flutter para dispositivos Android.  
Permite la interacción con arduinos, gestión de sesiones y transferencia de datos

Funcionalidades principales:
- Conexión con dispositivos Arduino
- Visualización de sesiones activas y totales
- Transferencia de datos entre backend y dispositivo
- Interfaz optimizada para uso móvil

Documentación:
La documentación específica del proyecto Android se encuentra en: android_front_upm\README.md


## 2. Windows Front (windows_front_upm)

Aplicación de escritorio desarrollada para Windows.  
Está enfocada en la administración completa del sistema, gestión de dispositivos y control de usuarios.

 Funcionalidades principales:
- Panel de administración de dispositivos
- Creación, eliminación y edición de dispositivos
- Gestión de sesiones y recargas
- Historial de actividad por dispositivo

La documentación específica del proyecto Windows se encuentra en: windows_front_upm\README.md


## Notas

Cada proyecto es independiente y contiene sus propias dependencias, configuración y documentación detallada en su respectivo directorio
